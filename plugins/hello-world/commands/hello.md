# Hello Command

A friendly greeting command that says hello to the user.

## Instructions

1. If the user provides a name argument, greet them by name
1. If no name is provided, use "World" as the default name
1. Output the greeting in a friendly, conversational tone

## Examples

- `/hello` → "Hello, World!"
- `/hello Claude` → "Hello, Claude!"
